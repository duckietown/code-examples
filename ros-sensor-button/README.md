# Example: ROS - Sensor - Power Button

This example shows how to detect the user input from a robot's power button in ROS.

## 1. Build

Build this project using the command,

```shell
dts devel build
```

## 2. Run

Run the project using the command,

```shell
dts devel run -R [ROBOT_NAME]
```

**NOTE:** Make sure to replace `[ROBOT_NAME]` with the name of the robot you are using in the commands above.


### Expected Result

You should see the type of event detected from the power button printed to the console.
